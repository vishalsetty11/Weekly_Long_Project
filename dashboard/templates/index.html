🏗️ Medallion Pipeline Stages

RAW

Daily Bhavcopy files are fetched from the NSE website and stored dynamically inside MotherDuck.

BRONZE

Immutable data ingestion layer. Raw data is scrubbed using strict regex filters to remove ETFs, mutual funds, and non-equity indices.

SILVER

Data transformation layer. This stage executes a recursive CTE framework to clean data types, calculate smoothed tracking columns, and filter for delivery percentages over 50%.

GOLD

High-conviction analytical snapshot layer. Merges price breakouts and volume surges into optimized historical views for the dashboard.

📊 Metric Columns Explained

Symbol The NSE stock ticker. Clicking it routes directly to its corresponding live technical chart on TradingView.
Close Price The last traded close value for the given weekly timeline block. Clicking it opens the asset's fundamentals profile on Screener.in.
RSI (14) Relative Strength Index computed using Wilder's Exponential Smoothing over a 14-period lookback. Only assets with a value greater than 45 pass the layout filter.
50-week MA A true Exponential Moving Average (EMA) capturing structural long-term trend direction. Color turns green when Close Price actively tracks above it.
180D High The peak weekly close reached during the trailing 26-week window context (~6 months), serving as the baseline threshold to confirm a price breakout.
3X Vol Mult The current weekly total trading volume relative to its 180-day historical weekly average. Signifies institutional accumulation when it triggers a surge.
